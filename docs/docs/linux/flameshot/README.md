# Flameshot GUI

## Para adicionalo como atalho de teclado

Geralmente no ubuntu após a instalação ele não tem um atalho de teclado padrão.
Logo, eu gosto de configrar ele, mas sempre tem o problema de Wayland. Dai ele não inicia corretamente.
Pra contornar esse problema, o que se é necessáiro fazer no comando de atalho é fazer o seguinte, o adicione dessa maneira aqui:

```bash
sh -c "flameshot gui"
```
assim ele vai iniciar corretamente.
